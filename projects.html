ZooReact: Built with React Native.

ZooReact: an app which utilizes Zooniverse's API

Panoptes API documentation

This app uses Fetch API to get Zooniverse data about projects, collections, users. Zooniverse API, or, Panoptes, enables the client to fetch the data from either the staging endpoint or production endpoint. This app fetches data from the production endpoint, therefore displaying, for example: real projects and real users.

Lingo Chat: made for iOS

Lingo Chat: chat app for people around the world

Let everyone understand you!

The idea for this app came from my own language-learning experience. Whenever I would chat with a person who speaks another language, but with whom I don't have any common language, it becomes really hard to communicate. I thought it would be a good idea to make an app which lets you translate messages on the fly, since this is possible to do today. By utilizing already great translation API services, such as IBM's Watson translate, Yandex, or Google's translation API, client is able to convert incoming/outgoing messages to whatever language he/she would like. If, for example, I am texting a German native speaker who finds it easier to type in German than in English, I am able to set all of my incoming messages to be translated to English, or he/she can set all of his/her outgoing messages to be translated to English. You get the idea.

I believe that this app will help people who are passionate about learning a new language, like myself, and if nothing, at least ease the communication between people.

If you would like to try it out yourself, here is the link to the project: Lingo Chat

Kwowledge: made for Android

Kwowledge: app for WoW fans!

Learn more about World of Warcraft...

This app uses Battle.net API to get various information about WoW. It fetches data asynchronously by using RxJava, as well as OAuth 2.0 authentication to log the user to his/her World of Warcraft account. For example: you can obtain game information about bosses, quests, zones, other people's characters, etc... You can also see log in to your Blizzard WoW account to get interesting statistics about your own characters, such as: how much dmg you've dealt with a character, how many PvPs did you go to, wins/losses, etc...

Basically, this app lets you see all kinds of information related to the game and/or your character(s).

If you would like to try it out yourself, here is the link to the project: Kwowledge
